NO-EMOJI POLICY (Règle Centrale)
===============================

Interdictions (code & config):
- Pas d'emoji dans .bat, .js, .ts, .tsx, .json, logs, messages d'erreur.
- Pas d'emoji dans commentaires de code.

Exceptions limitées:
- Markdown documentation (.md) & README: tolérés avec parcimonie.

Rappels style:
- Messages: sobres, factuels.
- Scripts batch: sans décoration.
- Logs debug: texte brut.

Motifs:
- Compat consoles / CI
- Lisibilité durable
- Demande explicite propriétaire

Si violation détectée → corriger dans le même commit ou revert.

Pour le reste des consignes LLM: voir `handbook/LLM-ENTRYPOINT.md`.
